    11мһҘ. Pipeline Parallelism¶

    Pipeline ParallelismмқҖ лӘЁлҚёмқҳ layerлҘј м—¬лҹ¬ GPUм—җ лӮҳлҲ„м–ҙ л°°м№ҳн•ҳлҠ” л°©мӢқмқҙлӢӨ. Tensor Parallelмқҙ layer лӮҙл¶Җмқҳ н–үл ¬мқ„ лӮҳлҲҲлӢӨл©ҙ, Pipeline ParallelмқҖ layer л¬¶мқҢмқ„ лӮҳлҲҲлӢӨ. м•һ GPUмқҳ м¶ңл Ҙмқҙ лӢӨмқҢ GPUмқҳ мһ…л Ҙмқҙ лҗҳлҠ” кө¬мЎ°лӢӨ.

    н•ҷмҠө лӘ©н‘ң¶

    • Pipeline Parallelмқҙ layerлҘј лӮҳлҲ„лҠ” л°©мӢқмһ„мқ„ мқҙн•ҙн•ңлӢӨ.
    • Micro-batchмҷҖ pipeline bubbleмқҳ мқҳлҜёлҘј нҢҢм•…н•ңлӢӨ.
    • PPк°Җ TP, DPмҷҖ мЎ°н•©лҗҳлҠ” мқҙмң лҘј м„ӨлӘ…н•ңлӢӨ.

    11.1 Pipeline Parallelмқҳ м •мқҳ¶

    LLMмқҖ м—¬лҹ¬ Transformer layerк°Җ мҲңм„ңлҢҖлЎң мҢ“мқё кө¬мЎ°лӢӨ. Pipeline ParallelмқҖ мқҙ layerл“Өмқ„ м—¬лҹ¬ stageлЎң лӮҳлҲ„кі , к°Ғ stageлҘј лӢӨлҘё GPUм—җ л°°м№ҳн•ңлӢӨ. мҳҲлҘј л“Өм–ҙ 60 layer лӘЁлҚёмқ„ 4к°ң stageлЎң лӮҳлҲ„л©ҙ GPUл§ҲлӢӨ 15 layerм”© лӢҙлӢ№н•  мҲҳ мһҲлӢӨ.

    11.1.1 лӘЁлҚёмқҳ layerлҘј м—¬лҹ¬ GPUм—җ лӮҳлҲҲлӢӨ¶

    GPU 0мқҖ м•һмӘҪ layer, GPU 1мқҖ мӨ‘к°„ layer, GPU 2лҠ” л’ӨмӘҪ layerлҘј лӢҙлӢ№н•ңлӢӨ. мһ…л Ҙ tokenмқҳ hidden stateлҠ” GPU 0м—җм„ң мІҳлҰ¬лҗң л’Ө GPU 1лЎң м „лӢ¬лҗҳкі , лӢӨмӢң GPU 2лЎң м „лӢ¬лҗңлӢӨ.

    мқҙ л°©мӢқмқҖ weightлҘј layer лӢЁмң„лЎң лӮҳлҲ„лҜҖлЎң к°Ғ GPUк°Җ м „мІҙ лӘЁлҚё weightлҘј к°Җм§Ҳ н•„мҡ”к°Җ м—ҶлӢӨ. л§Өмҡ° к№ҠмқҖ лӘЁлҚёмқҙлӮҳ нҒ° лӘЁлҚёмқ„ м—¬лҹ¬ GPUм—җ лӮҳлҲ„м–ҙ мҳ¬лҰҙ л•Ң мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

    11.1.2 м•һ GPUмқҳ м¶ңл Ҙмқҙ лӢӨмқҢ GPUмқҳ мһ…л Ҙмқҙ лҗңлӢӨ¶

    Pipeline кө¬мЎ°м—җм„ңлҠ” stage к°„ мқҳмЎҙм„ұмқҙ л¶„лӘ…н•ҳлӢӨ. л’Ө stageлҠ” м•һ stageмқҳ м¶ңл Ҙмқҙ мһҲм–ҙм•ј кі„мӮ°мқ„ мӢңмһ‘н•  мҲҳ мһҲлӢӨ. л”°лқјм„ң лӢЁмқј micro-batchл§Ң мІҳлҰ¬н•ңлӢӨл©ҙ GPUл“Өмқҙ м°ЁлЎҖлЎң мқјн•ҳкі , мқјл¶Җ GPUлҠ” кё°лӢӨлҰ¬кІҢ лҗңлӢӨ.

    мқҙ кё°лӢӨлҰјмқ„ мӨ„мқҙкё° мң„н•ҙ micro-batchлҘј мӮ¬мҡ©н•ңлӢӨ. м—¬лҹ¬ micro-batchк°Җ pipeline м•Ҳмқ„ нқҗлҘҙл©ҙ, м•һ stageк°Җ лӢӨмқҢ micro-batchлҘј мІҳлҰ¬н•ҳлҠ” лҸҷм•Ҳ л’Ө stageлҠ” мқҙм „ micro-batchлҘј мІҳлҰ¬н•  мҲҳ мһҲлӢӨ.

    11.1.3 лӘЁлҚё к№Ҡмқҙ л°©н–Ҙ лі‘л ¬нҷ”лӢӨ¶

    Pipeline ParallelмқҖ лӘЁлҚёмқҳ к№Ҡмқҙ л°©н–Ҙмқ„ лӮҳлҲҲлӢӨ. мқҙ м җмқҙ Tensor Parallelкіј к°ҖмһҘ нҒ° м°ЁмқҙлӢӨ. TPлҠ” к°ҷмқҖ layer м•Ҳмқҳ кі„мӮ°мқ„ лӮҳлҲ„кі , PPлҠ” layer мҲңм„ңлҘј м—¬лҹ¬ GPUм—җ лӮҳлҲҲлӢӨ.

    лӘЁлҚёмқҙ л§Өмҡ° к№Ҡкі  layer лӢЁмң„лЎң лӮҳлҲ„кё° мү¬мҡё л•Ң PPк°Җ мқҳлҜёлҘј к°–лҠ”лӢӨ.

    11.2 Pipeline Parallelмқҳ м§ҒкҙҖ¶

    PipelineмқҖ кіөмһҘ мЎ°лҰҪ лқјмқёкіј л№„мҠ·н•ҳлӢӨ. к°Ғ мһ‘м—…мһҗк°Җ м ңн’Ҳмқҳ н•ң лӢЁкі„лҘј л§Ўкі , м ңн’Ҳмқҙ мҲңм„ңлҢҖлЎң лӢӨмқҢ мһ‘м—…мһҗлЎң мқҙлҸҷн•ңлӢӨ. мһ‘м—…мһҗк°Җ лӘЁл‘җ л°”мҒҳкІҢ мқјн•ҳл Өл©ҙ м—¬лҹ¬ м ңн’Ҳмқҙ лқјмқём—җ мҳ¬лқјмҷҖ мһҲм–ҙм•ј н•ңлӢӨ.

    11.2.1 кіөмһҘ мЎ°лҰҪ лқјмқёмІҳлҹј layerлҘј нҶөкіјн•ңлӢӨ¶

    мһ…л ҘмқҖ GPU 0мқҳ layerлҘј нҶөкіјн•ң л’Ө GPU 1лЎң мқҙлҸҷн•ҳкі , лӢӨмӢң GPU 2лЎң мқҙлҸҷн•ңлӢӨ. к°Ғ GPUлҠ” мһҗмӢ мқҙ л§ЎмқҖ layerл§Ң кі„мӮ°н•ңлӢӨ. мқҙ кө¬мЎ°лҠ” м§ҒкҙҖм Ғмқҙм§Җл§Ң, stage к°„ лҚ°мқҙн„° м „лӢ¬мқҙ н•„мҡ”н•ҳлӢӨ.

    л…ёл“ң к°„ нҶөмӢ мқҙ лҠҗлҰ° нҷҳкІҪм—җм„ңлҠ” TPмІҳлҹј layerл§ҲлӢӨ мһҗмЈј нҶөмӢ н•ҳлҠ” л°©мӢқліҙлӢӨ PPк°Җ лӮҳмқ„ мҲҳ мһҲлӢӨ. PPлҠ” stage кІҪкі„м—җм„ң activationмқ„ м „лӢ¬н•ҳлҠ” л°©мӢқмқҙкё° л•Ңл¬ёмқҙлӢӨ.

    11.2.2 Micro-batchк°Җ pipelineмқ„ мұ„мҡҙлӢӨ¶

    Pipelineмқ„ нҡЁмңЁм ҒмңјлЎң м“°л Өл©ҙ м—¬лҹ¬ micro-batchк°Җ н•„мҡ”н•ҳлӢӨ. Micro-batch Aк°Җ GPU 1лЎң л„ҳм–ҙк°Җл©ҙ GPU 0мқҖ micro-batch BлҘј мІҳлҰ¬н•  мҲҳ мһҲлӢӨ. мӢңк°„мқҙ м§ҖлӮҳл©ҙ м—¬лҹ¬ GPUк°Җ лҸҷмӢңм—җ м„ңлЎң лӢӨлҘё micro-batchлҘј мІҳлҰ¬н•ңлӢӨ.

    м¶”лЎ  servingм—җм„ңлҠ” н•ҷмҠөм—җм„ңмқҳ micro-batchмҷҖ лҸҷмһ‘ л§ҘлқҪмқҙ лӢӨлҘҙм§Җл§Ң, pipeline stageлҘј лҶҖлҰ¬м§Җ м•Ҡкё° мң„н•ҙ м—¬лҹ¬ мһ‘м—…мқ„ кІ№міҗм•ј н•ңлӢӨлҠ” мӣҗлҰ¬лҠ” к°ҷлӢӨ.

    11.2.3 Pipeline bubbleмқҙ мғқкёё мҲҳ мһҲлӢӨ¶

    Pipelineмқҙ мӢңмһ‘лҗ  л•ҢлҠ” л’ӨмӘҪ GPUк°Җ м•„м§Ғ л°ӣмқ„ мһ‘м—…мқҙ м—Ҷм–ҙ мү°лӢӨ. лҒқлӮ  л•ҢлҠ” м•һмӘҪ GPUк°Җ лҚ” мқҙмғҒ мғҲ мһ‘м—…мқҙ м—Ҷм–ҙ мү°лӢӨ. мқҙлҹ° л№„м–ҙ мһҲлҠ” мӢңк°„мқҙ pipeline bubbleмқҙлӢӨ. Micro-batch мҲҳк°Җ м Ғмңјл©ҙ bubble л№„мңЁмқҙ м»Өм ё нҡЁмңЁмқҙ л–Ём–ҙм§„лӢӨ.

    LLM м¶”лЎ м—җм„ңлҠ” мҡ”мІӯ кёёмқҙк°Җ лӢӨм–‘н•ҳкі  decodeк°Җ мҲңм°Ём Ғмқҙкё° л•Ңл¬ём—җ PP schedulingмқҙ к№ҢлӢӨлЎңмҡё мҲҳ мһҲлӢӨ. лӢЁмҲңнһҲ layerлҘј лӮҳлҲ„лҠ” кІғл§ҢмңјлЎң лҶ’мқҖ нҡЁмңЁмқҙ ліҙмһҘлҗҳм§Җ м•ҠлҠ”лӢӨ.

    11.3 Pipeline Parallelмқҳ мӮ¬мҡ© мғҒнҷ©¶

    Pipeline ParallelмқҖ лӢЁлҸ…мңјлЎңлҸ„ м“°мқҙм§Җл§Ң, лҢҖк·ңлӘЁ лӘЁлҚём—җм„ңлҠ” TPмҷҖ н•Ёк»ҳ мЎ°н•©лҗҳлҠ” кІҪмҡ°к°Җ л§ҺлӢӨ. н•ң stage м•Ҳм—җм„ңлҠ” TPлЎң layer лӮҙл¶Җ кі„мӮ°мқ„ лӮҳлҲ„кі , stage к°„м—җлҠ” PPлЎң layerлҘј лӮҳлҲ„лҠ” мӢқмқҙлӢӨ.

    11.3.1 лӘЁлҚёмқҙ л§Өмҡ° к№ҠмқҖ кІҪмҡ°¶

    Layer мҲҳк°Җ л§ҺмқҖ лӘЁлҚёмқҖ layer л¬¶мқҢмқ„ stageлЎң лӮҳлҲ„кё° мүҪлӢӨ. к°Ғ GPUк°Җ мқјл¶Җ layerл§Ң к°Җм§ҖлҜҖлЎң weight л©”лӘЁлҰ¬лҘј л¶„мӮ°н•  мҲҳ мһҲлӢӨ. лӢЁмқј GPUм—җ лӘЁлҚё м „мІҙлҘј мҳ¬лҰҙ мҲҳ м—ҶлҠ” кІҪмҡ° PPк°Җ м„ нғқм§Җм—җ л“Өм–ҙк°„лӢӨ.

    11.3.2 л…ёл“ң к°„ нҶөмӢ мқҙ лҠҗлҰ° кІҪмҡ°¶

    TPлҠ” layerл§ҲлӢӨ мһҰмқҖ collective communicationмқҙ н•„мҡ”н•ҳлӢӨ. л…ёл“ң к°„ л„ӨнҠёмӣҢнҒ¬к°Җ лҠҗлҰ¬лӢӨл©ҙ TPлҘј л…ёл“ң мӮ¬мқҙлЎң нҷ•мһҘн•ҳкё° м–ҙл өлӢӨ. PPлҠ” stage кІҪкі„м—җм„ң activationмқ„ л„ҳкё°лҠ” л°©мӢқмқҙлҜҖлЎң, мқјл¶Җ мғҒнҷ©м—җм„ңлҠ” л…ёл“ң к°„ нҷ•мһҘм—җ лҚ” м Ғн•©н•  мҲҳ мһҲлӢӨ.

    лӢӨл§Ң activation нҒ¬кё°мҷҖ scheduling л№„мҡ©лҸ„ кі л Өн•ҙм•ј н•ңлӢӨ. PPк°Җ н•ӯмғҒ л…ёл“ң к°„ нҶөмӢ  л¬ём ңлҘј н•ҙкІ°н•ҳлҠ” кІғмқҖ м•„лӢҲлӢӨ.

    11.3.3 TPмҷҖ н•Ёк»ҳ мЎ°н•©н•ҳлҠ” кІҪмҡ°¶

    лҢҖк·ңлӘЁ servingм—җм„ңлҠ” лі‘л ¬нҷ” м°Ёмӣҗмқ„ мЎ°н•©н•ңлӢӨ. мҳҲлҘј л“Өм–ҙ н•ң л…ёл“ң м•Ҳмқҳ 8 GPUлҠ” TPлЎң л¬¶кі , м—¬лҹ¬ л…ёл“ңм—җ кұёміҗ PP stageлҘј кө¬м„ұн•  мҲҳ мһҲлӢӨ. м—¬кё°м—җ replicaлҘј м—¬лҹ¬ к°ң л‘җл©ҙ DPк№Ңм§Җ мЎ°н•©лҗңлӢӨ.
