    소스 정책¶

    AI infrastructure 도구와 논문은 빠르게 바뀐다. 이 책은 공개 본문에 넣는 도구 동작, 수치, 옵션, benchmark 결과를 가능한 한 1차 출처 기준으로 확인한다.

    우선순위¶

    1. 공식 문서
    2. 공식 repository
    3. vendor documentation
    4. 원 논문 또는 arXiv paper
    5. project-hosted blog or release note
    6. survey, lecture note, third-party article

    • current quantization support and vLLM compatibility

    공개하지 않는 것¶

    이 웹북에는 개인 작업 로그, raw source files, prompt log 전체를 공개하지 않는다. 내부 메모는 구조 설계와 읽기 순서에만 사용하고, 공개 본문에는 검증된 외부 source를 기준으로 반영한다.
